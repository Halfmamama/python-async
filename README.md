# Примеры кода на Python и asyncio

В этом репозитории собраны примеры моего кода, демонстрирующие навыки разработки асинхронных сетевых приложений, многопоточных утилит и инструментов автоматизации браузера на Python. Все проекты и модули полностью самодостаточны, очищены от персональных данных и готовы к запуску.

## Содержимое репозитория

* **[Steam Market Monitor](./steam-market-monitor/)** — фоновый консольный демон для отслеживания цен лотов на торговой площадке Steam с отправкой уведомлений в Telegram.
* **[Async Networking & Automation Toolkit](./async-toolkit/)** — набор универсальных переиспользуемых модулей для асинхронного логирования, проверки прокси и повышения стабильности работы Playwright.
* **[Crypto Airdrop Automation](./crypto-airdrop-automation/)** — срез архитектуры фреймворка для параллельной мультиаккаунт-автоматизации взаимодействия с крипто-проектами и веб-кошельками через Playwright.
