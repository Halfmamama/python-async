# Async Networking & Automation Toolkit

Набор вспомогательных модулей для повышения стабильности сетевых запросов и браузерной автоматизации на Playwright.

## Содержимое

1. **`proxy_health.py`** — асинхронный инструмент для мониторинга состояния прокси. Выполняет запросы через `asyncio.to_thread` к двум независимым хостам. Отличает ошибки самого туннеля (ERR_TUNNEL_CONNECTION_FAILED и др.) от задержек чтения со стороны целевого сайта.
2. **`resilience.py`** — расширение для Playwright. Включает автоматическое снятие скриншота экрана и дамп исходного HTML в `logs/failures/` при перехвате исключений, безопасный клик по элементам (`click_if_visible` с логом вместо падения) и менеджер включения/выключения трассировки Playwright (tracing).
3. **`logger.py`** — централизованный логгер с фильтрацией чувствительных данных. Находит и заменяет в логах пароли, прокси-креды и токены Telegram на строку `[REDACTED]`.
4. **`result.py`** — типизированный контейнер `TaskResult` на базе dataclass для структурирования результатов выполнения задач в конкурентных процессах.

## Стек технологий
* **Python 3.11+**
* **Playwright** (async API)
* **Requests**

## Как запустить примеры
1. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   playwright install chromium
   ```
2. Запустить пример чекера прокси:
   ```bash
   python examples/check_proxy.py
   ```
3. Запустить пример отказоустойчивости Playwright:
   ```bash
   python examples/resilient_click.py
   ```
